# MuscleHub

## Overview

MuscleHub is a responsive fitness landing page designed to showcase a gym brand, training classes, educational blog content, and member engagement. Built with modern HTML, CSS, and JavaScript, it provides a polished online presence for a fitness club and helps users discover classes, services, and contact information.

## Features

- Responsive hero section with gym branding and call-to-action
- Mobile-friendly navigation menu with toggle support
- Class showcase section with progress bars and workout descriptions
- Blog feed section for fitness tips and latest news
- Newsletter signup form and contact information in the footer
- Smooth scrolling and back-to-top browsing experience

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts
- Ionicons

## Installation

1. Clone the repository.
2. Navigate to the project directory.
3. No additional dependencies are required.
4. Open `Index.html` in a browser or use a local development server.

## Usage

Open the site in a browser to explore the gym landing page. Use the navigation menu to jump between sections such as Home, About Us, Classes, and Blogs. The site includes interactive navigation toggling for mobile layouts, a newsletter signup form, and a back-to-top button for easy browsing.
